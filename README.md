# Piste Planner

A USA Fencing tournament schedule planner. Computes pool rounds, DE brackets, strip assignments, and referee allocations for multi-day tournaments (NACs, RYCs, SYCs, etc.).

https://noahlz.github.io/piste-planner

## Development

### Prerequisites

- Node.js 22+
- [pnpm](https://pnpm.io/)

### Commands

```bash
pnpm install
pnpm build
pnpm test
pnpm dev     # Run site locally
```

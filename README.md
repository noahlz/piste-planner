# Piste Planner

A USA Fencing tournament schedule planner. Computes pool rounds, DE brackets, strip assignments, and referee allocations for multi-day tournaments (NACs, RYCs, SYCs, etc.).

https://noahlz.github.io/piste-planner

## Development

### Prerequisites

- Node.js 22+
- [pnpm](https://pnpm.io/)

### Install Dependencies

```bash
pnpm install
```

### Run Dev Server

```bash
pnpm dev
```

### Run Tests

```bash
pnpm test
```

### Build

```bash
pnpm build
```

### Lint & Format

```bash
pnpm lint
pnpm format
```
